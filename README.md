## 排版不再"乱"

- **自定义规则**：正则表达式+预览+测试
- **批量处理**：多文件上传批量检查
- **规范检查**：中英文混排格式校验
- **修复建议**：自动提供修正方案

## 怎么用

打开 `checker.html` → 上传文件 → 查看检查结果
