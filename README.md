# Markdown 中文排版规范

[![HTML](https://img.shields.io/badge/HTML-standalone-orange)]()
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-CSS-blue)]()

Markdown 中文排版规范检查工具。

## 功能

- **自定义规则**：正则表达式 + 预览 + 测试
- **批量处理**：多文件上传批量检查
- **规范检查**：中英文混排格式校验
- **修复建议**：自动提供格式修正方案

## 使用

打开 checker.html 即可使用。
